# FinFlow – Financing Management Dashboard

FinFlow is a modern, premium, and minimalistic Financing Management Dashboard designed for business users managing customer financing, risk assessment, and payment collection workflows. 

Inspired by leading fintech SaaS platforms such as Stripe, Ramp, and Brex, FinFlow delivers an enterprise-grade user interface with smooth animations, dynamic responsive layout handling, and a Material Design 3 aesthetic.

---

## 🚀 Live Demo & Preview

The dashboard includes multiple fully functional mock views accessible via the sidebar navigation:
- **Dashboard Overview**: Key performance indicators, funnel metrics, and recent activities.
- **Applications**: Detailed tabular tracking of financing applications with dynamic filter chips and search functionality.
- **Payments**: Collections tracker, EMI schedules, and overdue alerts.
- **Customers**: Borrower management and verification tracker.
- **Loan Portfolio**: Portfolio distribution and risk segmentation analysis.
- **Credit Risk**: CIBIL scoring and portfolio risk scorecard.
- **Reports & AI Insights**: Downloadable summaries and actionable auto-reminders generated by FinFlow AI.
- **Compliance**: Regulatory checklist and audit tracking.
- **Settings**: Complete profile configuration (updating name, role, generating avatar initials, and toggling alert options).

---

## 🛠️ Tech Stack & Libraries

- **Core**: Semantic HTML5, Vanilla JavaScript (ES6+).
- **Styling**: Vanilla CSS3, custom CSS Variables (Material Design 3 spacing and color system).
- **Animations**: GSAP (GreenSock Animation Platform) & ScrollTrigger for fluid entrance effects and micro-interactions.
- **Charts & Visualizations**: Chart.js (for line, bar, donut, and area charts).
- **Icons**: Google Material Symbols.

## 📱 Responsive Layout System

FinFlow implements a responsive breakpoints system:
- **Desktop (1025px+)**: Fully expanded navigation drawer, horizontal timeline layout, and side-by-side dashboard cards.
- **Tablet (769px - 1024px)**: Collapsed navigation drawer (rail navigation) with interactive hover overlays.
- **Mobile (<= 768px)**: Hidden sidebar accessible via a header hamburger button, card-style table views, and responsive stacked grid columns.

*Note: The layout dynamically adapts and synchronizes state (expanding/collapsing/clearing overlays) when the viewport is resized across breakpoints without requiring a page refresh.*

## ⚙️ How to Run Locally

Since the dashboard uses vanilla JavaScript modules and loads assets dynamically, it should be served via a local web server.

### Option 1: Live Server (VS Code Extension)
1. Install the **Live Server** extension in Visual Studio Code.
2. Open the project folder in VS Code.
3. Click the **Go Live** button in the status bar at the bottom right.

### Option 2: Node.js (http-server / serve)
Run one of the following commands:
```bash
npx http-server -p 5500
# or
npx serve -l 5500
```

## ♿ Accessibility (WCAG AA Compliance)
- **Keyboard Navigation**: Interactive elements have distinct `:focus-visible` outline states.
- **Contrast**: Text elements meet WCAG AA contrast standards.
- **Semantic HTML**: Proper use of `<main>`, `<aside>`, `<nav>`, `<section>`, and `<header>` tags.
- **Screen Readers**: Active use of ARIA labels, roles (`role="navigation"`, `role="log"`, etc.), and `sr-only` class descriptions.
